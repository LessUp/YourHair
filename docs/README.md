## 🎯 快速链接

### 常见问题

#### Q: 如何开始使用？
A: 查看 [快速开始指南](./QUICK_START.md)，5分钟即可上手。

#### Q: 发型看起来不自然怎么办？
A: 查看 [混合模式使用指南](./BLEND_MODES_GUIDE.md)，学习如何调整透明度和混合模式。

#### Q: 如何添加自己的发型？
A: 查看 [自定义发型素材指南](./CUSTOM_HAIRSTYLES.md)，了解发型素材的制作和添加方法。

#### Q: 支持哪些浏览器？
A: 支持 Chrome 90+, Firefox 88+, Safari 14+, Edge 90+。详见 [FEATURES.md](../FEATURES.md#浏览器兼容性)。

#### Q: 照片会上传到服务器吗？
A: 不会！所有处理都在浏览器本地完成，照片不会上传到任何服务器。

## 💡 使用技巧

### 最佳实践
1. ✅ 使用正脸照片，光线充足
2. ✅ 透明度设置在 85% 左右
3. ✅ 优先尝试"叠加"或"柔光"混合模式
4. ✅ 先调整位置和大小，再调整融合效果
5. ✅ 使用 Canvas 模式导出最终图片

### 常见错误
1. ❌ 使用侧脸或低头照片
2. ❌ 透明度设置过低（<50%）
3. ❌ 不调整混合模式直接导出
4. ❌ 发型尺寸过大或过小
5. ❌ 使用模糊或低分辨率照片
