# 快速开始指南

## 5分钟上手 YourHair

### 第一步：准备照片 📸

选择一张符合以下要求的照片：
- ✅ 正脸照片（不要侧脸或低头）
- ✅ 光线充足，面部清晰
- ✅ 背景简洁（纯色背景最佳）
- ✅ 高清照片（建议至少 800x600 像素）
- ✅ JPG 或 PNG 格式

### 第二步：上传照片 ⬆️

1. 点击页面中央的"选择照片"按钮
2. 或者点击左上角的"上传照片"按钮
3. 从电脑中选择准备好的照片
4. 照片会立即显示在画布中

💡 **提示**: 照片仅在浏览器本地处理，不会上传到任何服务器！

### 第三步：选择发型 💇

1. 在右侧发型库中浏览可用的发型
2. 发型按类别分组：
   - 短发系列
   - 长发飘飘
   - 个性卷发
3. 点击任意发型即可应用到照片上

### 第四步：调整位置 🎯

1. **拖拽移动**: 直接用鼠标拖拽发型到合适位置
2. **缩放大小**: 使用右侧"缩放大小"滑块调整发型尺寸
3. **旋转角度**: 使用"旋转角度"滑块调整发型角度
4. **水平翻转**: 点击"水平翻转"按钮改变发型方向

💡 **技巧**: 先调整位置和大小，确保发型覆盖原发型区域

### 第五步：融合效果 ✨

这是最关键的一步，让发型看起来更自然：

1. **调整透明度**:
   - 拖动"透明度"滑块
   - 推荐值: 85%
   - 可以看到新旧发型的融合效果

2. **选择混合模式**:
   - 点击"混合模式"下拉菜单
   - 推荐模式:
     - 深色发型 → "正片叠底"
     - 浅色发型 → "滤色"
     - 通用效果 → "叠加"或"柔光"
   - 尝试不同模式，找到最自然的效果

💡 **快速设置**: 透明度85% + 叠加模式 = 适合大多数情况

### 第六步：高级模式（可选）🎨

想要更专业的效果？

1. 点击底部的"Canvas模式"按钮
2. 系统会使用Canvas API重新渲染
3. 效果更精细，适合最终导出

### 第七步：导出图片 💾

满意效果后：

1. 点击"导出预览图"按钮
2. 图片会以 2x 分辨率导出（更清晰）
3. 文件名: `yourhair-preview.png`
4. 保存到电脑，可以分享给发型师或朋友

## 常见场景

### 场景1: 想换个短发试试

```
1. 上传照片
2. 选择"短发系列"中的发型
3. 调整位置，确保覆盖原发型
4. 透明度: 85%
5. 混合模式: 正片叠底
6. 导出对比
```

### 场景2: 想染个浅色

```
1. 上传照片
2. 选择浅色发型（如金色大波浪）
3. 调整位置和大小
4. 透明度: 80%
5. 混合模式: 滤色
6. 导出效果图
```

### 场景3: 尝试个性造型

```
1. 上传照片
2. 选择"个性卷发"系列
3. 调整位置、旋转、翻转
4. 透明度: 85%
5. 混合模式: 叠加
6. 切换到Canvas模式
7. 导出高清图
```

## 最佳实践

### ✅ 推荐做法

1. 使用高质量正脸照片
2. 先调整位置和大小
3. 再调整透明度和混合模式
4. 尝试多种混合模式对比
5. 导出多个版本供选择

### ❌ 避免做法

1. 使用侧脸或低头照片
2. 使用模糊或低分辨率照片
3. 透明度设置过低（<50%）
4. 发型尺寸过大或过小
5. 不调整混合模式直接导出

## 故障排除

### 问题: 发型看起来不自然

**解决方案**:
1. 降低透明度到 70-85%
2. 切换混合模式（尝试"柔光"或"叠加"）
3. 调整发型位置，确保对齐
4. 检查照片质量和光线

### 问题: 发型颜色太淡

**解决方案**:
1. 提高透明度到 90-100%
2. 使用"正片叠底"或"颜色加深"模式
3. 切换到Canvas模式

### 问题: 发型颜色太深

**解决方案**:
1. 降低透明度到 60-75%
2. 使用"滤色"或"颜色减淡"模式
3. 调整发型大小

### 问题: 无法拖动发型

**解决方案**:
1. 确保已选择发型
2. 确保已上传照片
3. 刷新页面重试

## 进阶技巧

### 技巧1: 多角度对比

导出多个不同角度的效果图：
1. 调整旋转角度 -15°，导出
2. 调整旋转角度 0°，导出
3. 调整旋转角度 +15°，导出
4. 对比选择最佳角度

### 技巧2: 颜色对比

尝试同一发型的不同颜色效果：
1. 选择黑色短发，调整后导出
2. 选择蓝色短发，调整后导出
3. 对比颜色效果

### 技巧3: 长度对比

对比不同长度的发型：
1. 短发效果图
2. 中长发效果图
3. 长发效果图
4. 一次性看到所有可能性
