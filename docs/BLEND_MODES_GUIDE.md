# 混合模式使用指南

## 什么是混合模式？

混合模式决定了发型图层如何与底层照片融合。不同的混合模式会产生不同的视觉效果，选择合适的混合模式可以让发型看起来更自然。

## 混合模式详解

### 1. 正常 (Normal)
- **效果**: 直接覆盖，不进行混合
- **适用场景**: 发型完全替换原发型
- **推荐透明度**: 100%
- **最佳用途**: 背景简洁的照片

### 2. 正片叠底 (Multiply) ⭐ 推荐
- **效果**: 将两个图层的颜色相乘，结果更暗
- **适用场景**: 深色发型，保留头发的阴影和纹理
- **推荐透明度**: 80-90%
- **最佳用途**: 黑色、棕色等深色发型

### 3. 滤色 (Screen)
- **效果**: 与正片叠底相反，结果更亮
- **适用场景**: 浅色发型，提亮效果
- **推荐透明度**: 70-85%
- **最佳用途**: 金色、银色等浅色发型

### 4. 叠加 (Overlay) ⭐ 推荐
- **效果**: 结合正片叠底和滤色，增强对比度
- **适用场景**: 最常用的混合模式，效果自然
- **推荐透明度**: 85-95%
- **最佳用途**: 大多数发型，通用性强

### 5. 变暗 (Darken)
- **效果**: 保留两个图层中较暗的像素
- **适用场景**: 深色发型，保留暗部细节
- **推荐透明度**: 80-90%
- **最佳用途**: 需要保留原照片暗部的情况

### 6. 变亮 (Lighten)
- **效果**: 保留两个图层中较亮的像素
- **适用场景**: 浅色发型，提亮效果
- **推荐透明度**: 75-85%
- **最佳用途**: 染浅色的效果

### 7. 颜色减淡 (Color Dodge)
- **效果**: 提亮底层颜色，产生强烈的高光效果
- **适用场景**: 特殊效果，高光发型
- **推荐透明度**: 50-70%
- **最佳用途**: 金属色、高光效果

### 8. 颜色加深 (Color Burn)
- **效果**: 加深底层颜色，增加饱和度
- **适用场景**: 深色发型，增强色彩
- **推荐透明度**: 60-80%
- **最佳用途**: 需要强烈色彩的发型

### 9. 柔光 (Soft Light) ⭐ 推荐
- **效果**: 柔和的叠加效果，比overlay更温和
- **适用场景**: 自然融合，柔和效果
- **推荐透明度**: 85-95%
- **最佳用途**: 追求自然效果的发型

### 10. 强光 (Hard Light)
- **效果**: 强烈的叠加效果，对比度高
- **适用场景**: 需要强烈对比的发型
- **推荐透明度**: 75-90%
- **最佳用途**: 个性化、夸张的发型效果

## 快速选择指南

### 按发型颜色选择

| 发型颜色 | 推荐混合模式 | 透明度 |
|---------|------------|--------|
| 黑色/深棕 | 正片叠底 (Multiply) | 85% |
| 浅棕/棕色 | 叠加 (Overlay) | 90% |
| 金色/黄色 | 滤色 (Screen) | 80% |
| 红色/紫色 | 柔光 (Soft Light) | 85% |
| 银色/灰色 | 叠加 (Overlay) | 85% |
| 彩色/特殊色 | 叠加 (Overlay) | 80% |

### 按效果需求选择

| 需求 | 推荐混合模式 |
|------|------------|
| 最自然的效果 | 柔光 (Soft Light) |
| 保留阴影细节 | 正片叠底 (Multiply) |
| 提亮发色 | 滤色 (Screen) |
| 增强对比度 | 叠加 (Overlay) |
| 完全替换 | 正常 (Normal) |

## 调整技巧

1. **先选混合模式，再调透明度**: 不同混合模式需要不同的透明度
2. **对比多种模式**: 尝试3-4种混合模式，选择最自然的
3. **微调透明度**: 在推荐范围内上下调整5-10%
4. **考虑光线**: 照片光线充足时，可以降低透明度；光线暗时，提高透明度
5. **保存对比图**: 导出多个版本，对比选择最佳效果

## 常见问题

### Q: 为什么发型看起来不自然？
A: 尝试以下方法：
- 降低透明度到70-85%
- 切换到"柔光"或"叠加"模式
- 调整发型位置和大小

### Q: 发型颜色太淡/太深？
A: 
- 太淡：提高透明度，或使用"正片叠底"模式
- 太深：降低透明度，或使用"滤色"模式

### Q: 如何获得最专业的效果？
A: 
1. 使用高质量的正脸照片
2. 选择"叠加"或"柔光"模式
3. 透明度设置在85%左右
4. 切换到Canvas模式进行最终渲染
5. 导出2x分辨率的图片

## 实战案例

### 案例1: 深色短发
- 混合模式: 正片叠底 (Multiply)

- 效果: 保留头发的自然阴影和纹理

### 案例2: 金色长发
- 混合模式: 滤色 (Screen)
- 透明度: 80%
- 效果: 明亮自然的金发效果

### 案例3: 紫色波波头
- 混合模式: 叠加 (Overlay)
- 透明度: 85%
- 效果: 色彩鲜艳但不失自然

### 案例4: 银灰造型
- 混合模式: 柔光 (Soft Light)
- 透明度: 90%
- 效果: 柔和的银灰色，时尚感强
