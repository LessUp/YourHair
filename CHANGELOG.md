# 更新日志 Changelog

## [2.0.0] - 2024-12-08

### ✨ 新增功能

#### 高级图像融合系统
- **透明度控制**: 添加发型透明度调整滑块（0-100%）
- **混合模式**: 支持10种专业混合模式
  - 正常 (Normal)
  - 正片叠底 (Multiply)
  - 滤色 (Screen)
  - 叠加 (Overlay)
  - 变暗 (Darken)
  - 变亮 (Lighten)
  - 颜色减淡 (Color Dodge)
  - 颜色加深 (Color Burn)
  - 柔光 (Soft Light)
  - 强光 (Hard Light)

#### 双渲染引擎
- **CSS模式**: 使用CSS `mix-blend-mode` 实现快速预览
- **Canvas模式**: 使用Canvas API实现专业级融合效果
- 支持一键切换渲染模式

#### 用户体验改进
- 添加使用技巧面板，显示最佳实践
- 优化拖拽交互，支持触摸设备
- 改进控制面板UI，更直观的参数调整
- 添加实时参数显示（百分比、角度等）

### 📚 文档完善
- 新增快速开始指南 (QUICK_START.md)
- 新增混合模式使用指南 (BLEND_MODES_GUIDE.md)
- 新增自定义发型素材指南 (CUSTOM_HAIRSTYLES.md)
- 更新 README 文档，添加详细使用说明

### 🎨 UI/UX 优化
- 重新设计控制面板，更现代化的界面
- 添加混合模式选择器
- 优化滑块样式和交互反馈
- 添加模式切换按钮

### 🔧 技术改进
- 创建 AdvancedCanvas 组件，支持Canvas渲染
- 优化图像处理性能
- 改进代码结构和组件复用
- 添加 TipsPanel 组件

### 🐛 Bug修复
- 修复拖拽时的性能问题
- 优化图像导出质量
- 修复移动端触摸事件
