# YourHair 快速参考卡片

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问应用
http://localhost:5173
```

## 📖 5步上手

1. **上传照片** → 点击"上传照片"按钮
2. **选择发型** → 从右侧发型库选择
3. **调整位置** → 拖拽发型到合适位置
4. **融合效果** → 调整透明度85% + 选择"叠加"模式
5. **导出图片** → 点击"导出预览图"

## 🎨 推荐设置

### 深色发型
```
透明度: 85%
混合模式: 正片叠底 (multiply)
```

### 通用设置

透明度: 85%
混合模式: 叠加 (overlay)

## 🔧 常见问题

### Q: 发型不自然？
A: 降低透明度到70-85%，使用"柔光"或"叠加"模式

### Q: 颜色太淡？
A: 提高透明度，使用"正片叠底"模式

### Q: 颜色太深？
A: 降低透明度，使用"滤色"模式

### Q: 无法拖动？
A: 确保已上传照片并选择发型
